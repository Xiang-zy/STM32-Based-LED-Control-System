# STM32-Based-LED-Control-System
《微机系统与接口》期末大作业：单片机接入八个 LED、两个按键、一个串口以及模拟量采集信号 AD。实现如下功能：
1、 按间隔0.25秒，实现跑马灯（用定时器中断实现）；
2、按键KCHG为换向按键，每按一次， LED 跑马灯方向反向（用外部中断实现）；
3、按键KPAUS为暂停键，按一下停，再按一下继续跑（外部中断实现）； 
4、定时采集可调电阻PR的抽头信号AD；
5、按1秒定时，将AD值通过串口发送；
6、根据 AD值，采用PWM方式控制LED0-7的亮度，当AD小于10时为最暗，占比1%；大于 4080 时为亮度全开，占比100%；当AD值介于10和4080之间按线性调整亮度百分比，PWM通过场效应管的控制级LEDCMD来控制亮度。
